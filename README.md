Flipkart Data Insights

 Motivación

En un mercado digital tan competitivo como el de Flipkart, comprender cómo los clientes perciben los productos es clave para mejorar la experiencia de compra y fortalecer la confianza en la plataforma. Este proyecto nace con la intención de convertir datos en conocimiento útil, capaz de guiar decisiones estratégicas.

---

Propósito del proyecto

- Transformar datos crudos en información clara y visual.  
- Identificar patrones ocultos entre precio y calificación.  
- Ofrecer un marco de análisis que pueda ser replicado en otros contextos de e-commerce.  
- Servir como ejemplo educativo para mostrar cómo la analítica de datos puede generar valor real.

---

 Herramientas y entorno

- Lenguaje: Python 3.x  
- Entorno de trabajo: Jupyter Notebook en Visual Studio Code  
- Principales librerías:  
  - pandas → manejo de datos  
  - numpy → cálculos numéricos  
  - matplotlib y seaborn → visualización  

---

 Principales hallazgos

- Los productos de precio bajo concentran la mayor densidad de calificaciones.  
- No existe una relación lineal fuerte entre precio y satisfacción del cliente.  
- Los productos premium requieren un análisis más profundo para entender su posicionamiento.  

---

 Impacto social y empresarial

Este proyecto no solo aporta valor a empresas, sino también a comunidades educativas:  
- Empresas → pueden ajustar estrategias de precios y mejorar la gestión de catálogo.  
- Educadores → pueden usarlo como ejemplo práctico para enseñar análisis exploratorio.  
- Consumidores → se benefician indirectamente de mejores decisiones basadas en datos.
